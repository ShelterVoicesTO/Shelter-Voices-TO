Setting up a "GitHub" account & "GitHub Desktop" program:
====================================================
The following link is an excellent Utube tutorial for setting up what
YOU NEED to access the GitHub repository:
<a href="https://www.youtube.com/watch?v=8Dd7KRpKeaE">UTube GitHub Tutorial</a>

It will show you how to do the following:
1. Set up a personal account with GitHub
2. Download "GitHub Desktop" program:
- this will be your local clone of the repository since the
advantage of working with GitHub is that you can do all your work
offline and then upload the finished/edited files to the site using
the GitHub Desktop program
